# ProyectoVentas

## 📘 Descripción
Este proyecto corresponde a la **Entrega 1 (Semana 3)** del curso de **Conceptos Fundamentales de Programación** 

El objetivo de esta primera entrega es crear una clase en Java llamada **`GenerateInfoFiles`** que genera archivos de prueba (archivos de texto) con información de vendedores, productos y ventas.  
Estos archivos serán utilizados más adelante en el proyecto final para organizar y procesar los datos.

---

## 📂 Archivos generados
Al ejecutar la clase `GenerateInfoFiles` en Eclipse, el programa crea automáticamente:

- **vendedores.txt** → lista de vendedores con documento, nombre y apellido.  
- **productos.txt** → lista de productos con ID, nombre y precio.  
- **ventas_[id].txt** → archivo de ventas para cada vendedor.
